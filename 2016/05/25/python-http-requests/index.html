响应内容

http内容在传输时一般会压缩后传送，requests会对经过gzip和deflate压缩的内容进行自动解包，以方便我们获取响应内容进行处理，可以从response.content获取以字节形式返回的响应内容。

以上方法方便查看文本形式的响应内容，非文本的内容需进一步处理方能友好查看。

解析JSON内容

很多时候需处理响应的json内容，requests无需其他json库即可解析返回的json内容，非常方便。

POST表单

post方法可以传输一个表单数据至远端以处理，requests的post方法有一个data参数可以接收一个python字典作为表单数据。
