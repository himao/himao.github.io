Linux下批量替换目录下所有文件中的字符串

发表于 2017-01-03 | 更新于 2017-01-03 | 阅读次数

Linux下对目录中所有文件指定字符串批量替换可以使用sed和grep两个工具，配合精准高效的正则表达式来达到预期目的。

不包含子目录

待处理目录下不包含子目录的情况比较简单，直接使用sed即能达到目的。

包含子目录

如果待处理目录下包含子目录且需对子目录下的文件也进行替换时，需要借助grep工具先递归查找到所有包含需替换字符串的文件，再配合sed工具进行替换。

grep使用-l和-r参数，查找包含指定字符串的文件，查找范围为当前目录。-r参数表示递归查找，即查找当前目录及子目录，-l表示返回匹配的文件名，具体参数解释可以参考grep的手册。由此可得到所有需替换的文件，返回给sed进行替换处理。
